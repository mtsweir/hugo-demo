Typography

Component status
Stable

Body is a typography component primarily used within Bula components and for blocks of text.

Typeface

Bula's uses the open-source typface Nunito Sans. Nunito Sans can be downloaded from the official Google Fonts library.

Nunito Sans 400

Nunito Sans 600

Nunito Sans 700

Size

Body has a variety of sizes available with text-body-01 which will display content at the base font size.
